Password Strength Checker

DecodeLabs Cyber Security Internship – Project 1

------------------------------------------------------------

Project Overview

The Password Strength Checker is a Python-based application that evaluates the strength of a user's password using predefined security rules. The program checks password length, uppercase letters, numbers, and special characters before classifying the password as Weak, Medium, or Strong.

How to Run

1. Download or clone the repository.

2. Open the project in Visual Studio Code.

3. Run the following command:

python Password_Checker.py

4. Enter a password to check its strength.

Example

Input

Sameer@123

Output

Password contains an uppercase letter.

Password contains a number.

Password contains a special character.
